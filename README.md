## 🌟 Tentang Kami

**Fakultas Teknologi Bisnis dan Sains (FTBS)** adalah bagian dari Universitas Dharma Wacana yang berkomitmen mencetak generasi unggul dalam bidang teknologi, bisnis, dan sains. Kami mengintegrasikan pendidikan, penelitian, dan pengabdian masyarakat berbasis teknologi untuk menjawab tantangan masa depan.

🎓 Program Studi :
- Sistem Informasi
- Teknik Informatika
- Bisnis Digital
